집합을 비교할 수도 있겠다. 할 수 있는 비교 가운데 명백한 것으로는 다음을 꼽을 수 있다. 한 집합 안에 있는 모든 것이 다른 집합에도 있다. 새로운 표기법 도입 과정에 다분히 비중을 차지할 상황이다.

정의 2.5 (부분집합).

집합 $A$의 모든 구성원이 집합 $B$의 구성원이기도 할 때, $A$를 $B$dml 부분집합(subset)이라고 부르며, $A\subseteq B$라고 적는다. 만약 $A$가 $B$의 부분지밥이 아니라면 $A\nsubseteq B$라고 적는다. 만약 $A\subseteq B$지만 $A\neq B$라면 $A\subsetneq B$라고 적으며 $A$를 $B$의 진부분집합이라고 부른다.

예제 2.6.

모든 집합은 자기 자신의 부분집합이며, $\varnothing$은 모든 집합의 부분집합이고, 짝수 집합은 자연수 집합의 부분집합이다. 더불어 $\left \{ a, b \right \}\subseteq \left \{ a,b,c \right \}$이다. 다만 $\left \{ a,b,e \right \}$는 $\left \{ a,b,c \right \}$의 부분집합이 아니다.

예제 2.7.

$2$는 정수 집합의 구성원인데 반해 짝수 집합은 정수 집합의 부분집합이다. 하지만 한 집합은 어떤 다른 집합의 구성원인 동시에 부분집합일 수도 있다. 가령 $\left \{ 0 \right \}\in\left \{ 0, \left \{ 0 \right \} \right \}$인 동시에 $\left \{ 0 \right \}\subseteq\left \{ 0, \left \{ 0 \right \} \right \}$인 것이 가능하다.

외연성은 집합의 동일성 기준을 제공한다. $A$의 모든 구성원이 $B$의 구성원이기도 하며 그 역 또한 마찬가지일 때 그리고 오직 그런 경우(iff) $A=B$다. "부분집합"의 정의는 $A\subseteq B$를 정확히 이 기준의 전반부 절반에 따라 정의한다. $A$의 모든 구성원이 $B$의 구성원이기도 하다는 것. 물론 이 정의는 $A$와 $B$를 바꿔도 적용된다. 즉 $B$의 모든 구성원이 $A$의 구성원이기도 할 떄 그리고 오직 그런 경우 $B\subseteq A$다. 이는 곧 외연성에서 "역 또한 마찬가지"(vice versa)라는 부분과 정확히 맞물린다. 달리 말해 외연성은 집합들이 서로 부분집합인 경우 그리고 오직 그런 경우에만 집합들이 같다는 것을 함의한다.

명제 2.8.

$A\subseteq B$인 동시에 $B\subseteq A$일 때 그리고 오직 그런 경우 $A=B$다.

추가적인 표기법을 도입하기 좋은 기회다. $A$를 $B$의 부분집합이라고 정의하며 "$A$의 모든 구성원은 ..."이라고 말할 때 "..."를 "$B$의 구성원"이라고 채웠다. 다만 이는 표현의 일반적인 모양(shape)에 그치므로 형식적인 표기법의 소개가 도움이 되겠다.

정의 2.9.

$(\forallx \in A)\varphi$는 \forallx(x \in A \to \varphi)$를 축약한다. 이와 유사하게 $(\exists x \in A)\varphi$는 $(x \in A \land \varphi)$를 축약한다.

이 표기법을 사용함으로써 $(\forallx \in A)x \in B$이고 오직 그런 경우 $A \subseteq B%라고 말할 수 있다. 이어서 집합의 한 특정한 유형에 관해 생각하겠다. 이것은 주어진 한 집합의 모든 부분집합의 집합이다.

정의 2.10 (멱집합).

집합 $A$의 모든 부분집합을 포함하는 집합을 $A$의 멱집합(power set of $A$)이라고 부르며 $\wp(A)$라고 적는다.

예제 2.11.

\left \{ a, b, c \right \}의 모든 가능한 부분집합은 무엇인가? $\varnothing, \left \{ a\right \}, \left \{b \right \},\left \{c \right \},\left \{ a, b \right \},\left \{ a, c \right \} ,\left \{ b, c \right \},\left \{ a, b,c \right \}이다. 이 부분집합 전체의 집합은 $\wp(\left \{ a, b,c \right \})$이다.

$\wp(\left \{ a, b,c \right \})=\left \{\varnothing,\left \{ \left \{ a \right \},\left \{ b \right \},\left \{c \right \},\left \{ a, b \right \},\left \{ a, c \right \},\left \{ a, b,c \right \} \right \} \right \}$
